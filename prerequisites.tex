\section{Prerequisites}
\begin{enumerate}
  \item A local Git client installation.

  \item A local QuickSort product checkout at \verb|~/soup/quicksort|.

  \item Ruby, Rake, Amber, Autodoc, docbld, and tlc-article installed and
    available to the validation shell.

  \item The \verb|AMBERPATH|, \verb|AUTODOCPATH|, and \verb|DOCBLDPATH|
    environment variables must reference their corresponding product checkouts.

  \item The validation task must be run from the cloned \verb|git-client|
    product checkout and must be allowed to create and replace
    \verb|remote/quicksort.git| and local validation workspaces named
    \verb|package-*|.

\end{enumerate}
